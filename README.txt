#TR
Python dili kulalnılarak yazılmış basit bir Discord Botudur.
Kendiniz Özelleştirebilirsiniz tek yapmanız gereken main.py dosyasının içinde bulunan herhangi bir boşluğa komudunuzu yazmak

Gerekenler
python
visual studio code

bot u isterseniz main.py dosyasının içine girip terminali açıp "py main.py" yazarak başlatabilirsiniz.
istersenizde hazır Start.bat ile başlatabilirsiniz tek farkı biri daha kullanışlı diyeri daha şatafatlı renkli ve güzel.









#ing
It is a simple Discord Bot written using Python language.
You can customize it yourself, all you have to do is type your command in any space in the main.py files.

Requirements
python
visual studio code

If you want the bot, you can go into the main.py files and start the terminal with "py main.py".
If you want, you can start it with the ready-made Start.bat, the only difference is that one is more useful, the other is more flashy, colorful and beautiful.
